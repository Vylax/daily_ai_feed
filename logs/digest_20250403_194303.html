<h3>Example: Creating a basic LangGraph Node</h3>

<p>First, you'll need to install LangChain and LangGraph:</p>
<pre><code class="language-python">
# pip install langchain langgraph

<p>Now, let's create a simple LangGraph:</p>

<pre><code class="language-python">
from langgraph.graph import StateGraph, END
from langchain_core.runnables import chain

# Define a simple function for our node
def node_function(state):
    return {"output": f"Processed: {state['input']}"}

# Define the state (a simple dictionary in this case)
state = {"input": "Hello, LangGraph!"}

# Create the graph
graph = StateGraph(state)

# Add a node
graph.add_node("node1", node_function)

# Set the entrypoint
graph.set_entry_point("node1")

# Connect the node to the end
graph.add_edge("node1", END)

# Compile the graph
chain = graph.compile()

# Invoke the chain
result = chain.invoke(state)

# Print the result
print(result)
</code></pre>

<p><b>Explanation:</b></p>
<ul>
  <li>We import necessary modules from `langgraph` and `langchain_core`.</li>
  <li>`node_function` is a simple function that processes the state.</li>
  <li>We initialize the `StateGraph` with an initial state dictionary.</li>
  <li>We add a node named "node1" that runs the `node_function`.</li>
  <li>We set "node1" as the entry point of the graph.</li>
  <li>We connect "node1" to the `END` node, signifying the graph's termination point.</li>
  <li>Finally, we compile the graph into a runnable chain and invoke it.</li>
</ul>
